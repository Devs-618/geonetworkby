Стварэнне групы¶

The administrator can create new groups of users. User groups could correspond to logical units within an organisation, for example Fisheries, Agriculture, Land, Water, Health and so on.

To create new groups you should be logged on with an account that has administrative privileges.

  1. Выберыце ў меню кнопку Адміністраванне. На старонцы адміністравання абярыце Кіраванне групай.

  2. Выберыце Дадаць новую групу. Вы можаце выдаліць групу Sample;

  3. Запоўніце дэталі. Адрас электроннай пошты будзе выкарыстоўвацца для адпраўкі водгукаў аб спампоўках даных, калі яны адбываюцца для рэсурсаў, якія ўваходзяць у групу.

    Warning

    Імя НЕ павінна ўтрымліваць прабелы! Вы можаце выкарыстоўваць панэль лакалізацыі, каб даць лакалізаваныя імёны для груп.

  4. Click Save

Прывілеі доступу могуць быць устаноўлены для запісу метададзеных. Вы можаце вызначыць прывілеі для кожнай групы.

Прывілеі, якія можна задаць, тычацца бачнасці метададзеных (Апублікаваць), даных Спампаваць, доступу Інтэрактыўнай карты і адлюстравання запісу ў раздзеле Выбранае галоўнай старонкі.

Праўка вызначае групы, для якіх рэдактары могуць рэдагаваць запіс метададзеных.

Паведаміць вызначае, якія групы будуць апавешчаныя пры спампоўцы файла, якім кіруе GeoNetwork.

Ніжэй прыведзены прыклад табліцы кіравання прывілеямі, звязанай з наборам даных.
