Стварыце свой уласны выгляд¶

Калі вы аддаеце перавагу не рэдагаваць агульныя файлы па змаўчанні або калі вам патрэбны зусім іншы карыстальніцкі інтэрфейс, лепш за ўсё стварыць свой уласны карыстальніцкі выгляд.

Дадайце новы від¶

Як паказана ў папярэднім раздзеле, вам трэба будзе наладзіць некалькі файлаў для загрузкі новага прагляду карыстацкага інтэрфейсу.

  • стварыць новую тэчку ў web-ui/src/main/resources/catalog/views/.

  • дадайце templates/index.html у гэтую тэчку.

  • дадайце файл javascript, які загружае ваш асноўны модуль AngularJS

  • дадаць менш файлаў

Наладзьце свой выгляд¶

Дапусцім, мы хочам стварыць карыстальніцкае прадстаўленне з назвай &quot;геапартал&quot;.

  • Вам трэба будзе стварыць новую папку web-ui/src/main/resources/catalog/views/geoportal.

  • Дадайце новы файл web-ui/src/main/resources/catalog/views/geoportal/templates/index.html, які змяшчае

<div class="geoportal">Welcome to geoportal</div>

  • Дадайце новы файл web-ui/src/main/resources/catalog/views/geoportal/search.js, які змяшчае

(function() {

  goog.provide('gn_search_geoportal');
  goog.require('gn_search');

  var module = angular.module('gn_search_geoportal', ['gn_search']);

})();

Note

Вы павінны ўключыць модуль gn_search.

  • Дадайце новы файл web-ui/src/main/resources/catalog/geoportal/views/geoportal/less/gn_search_geoportal.less, які змяшчае

.geoportal {
    font-size: 36px;
    font-weight: bold;
}

Запусціце свой выгляд¶

Каб убачыць ваш новы выгляд, вы можаце запусціць службу catalog.search з новым выглядам параметраў: http://localhost:8080/geonetwork/srv/eng/catalog.search?view=geoportal

Or, go into admin settings page and set the system/ui/defaultView (User interface configuration -> Choose the user interface to use) to geoportal.
