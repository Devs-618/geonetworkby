Узбагаціце свой уласны выгляд¶

Цяпер мы стварылі нашу структуру з базавым кантэнтам, мы хочам стварыць сапраўдную старонку каталога.

Для гэтага вы можаце ўключыць змесціва прагляду па змаўчанні або стварыць уласнае прыкладанне на аснове асноўных кампанентаў GeoNetwork.

Кампаненты GeoNetwork¶

Дызайн карыстацкага інтэрфейсу заснаваны на агульных віджэтах, логіцы і кампанентах. У тэрмінах AngularJS мы гаворым пра «сэрвісы» і «дырэктывы».

Усе кампаненты захоўваюцца ў тэчцы `components <https://github.com/geonetwork/core-geonetwork/tree/develop/web-ui/src/main/resources/catalog/components> `_

  • Main page useful components

Галоўны пошукавы ўвод

<input type="text"
       class="form-control input-lg"
       id="gn-any-field"
       data-ng-model="searchObj.params.any"
       data-ng-model-options="modelOptionsForm"
       placeholder="{{'anyPlaceHolder' | translate}}"
       data-ng-keyup="$event.keyCode == 13 && triggerSearch()"
       data-typeahead="address for address in getAnySuggestions($viewValue)"
       data-typeahead-loading="anyLoading" class="form-control"
       data-typeahead-min-length="1"
       data-typeahead-focus-first="false">

Панэль граняў

<div data-ng-show="searchResults.records.length > 0"
     data-gn-facet-dimension-list="searchResults.dimension"
     data-params="searchObj.params"
     data-facet-type="facetsSummaryType"
     data-current-facets="currentFacets">
</div>

Інфармацыйны менеджэр абвестак

<div data-gn-alert-manager=""></div>

Спіс вынікаў

<div data-ng-show="searchResults.records.length > 0"
     data-gn-results-container=""
     data-search-results="searchResults"
     data-template-url="resultTemplate"
     data-map="searchObj.searchMap"></div>

Геаграфічны пошук

<div data-gn-map-field="searchObj.searchMap"
     data-gn-map-field-geom="searchObj.params.geometry"
     data-gn-map-field-opt="searchObj.mapfieldOption"
     class="gn-search-map">

Такім чынам, вы можаце паўторна выкарыстоўваць усе гэтыя кампаненты ў сваіх шаблонах, каб пабудаваць выгляд, які хочаце.

Стылі¶

Important

Вы павінны імпартаваць стыль па змаўчанні і дадаць уласныя правілы CSS замест таго, каб ствараць уласны з нуля.

У gn_search_geoportal.less імпартуйце менш файлаў па змаўчанні.

  • Імпарт агульных стыляў пошуку (калі вы пачынаеце новы выгляд з нуля)

@import "../../../style/gn_search.less";

  • Імпартаваць стылі пошуку прагляду па змаўчанні (калі вы заснуеце свой прагляд на аснове прагляду па змаўчанні)

@import "../../default/less/gn_search_default.less";

Затым у файле less вы можаце пачаць дадаваць новыя правілы

.gn-resultview .list-group-item {
    background-color: red;
    &:hover {
    background-color: orange;
    }
}
