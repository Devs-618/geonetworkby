Настройка відаў метададзеных¶

GeoNetwork дазваляе распрацоўшчыкам лёгка змяняць або дадаваць прагляд метададзеных. Карыстальнік можа змяніць выгляд у адпаведнасці са сваімі патрэбамі.

Па змаўчанні першапачатковым праглядам з&#39;яўляецца прагляд AngularJS вынікаў, вернутых службай пошуку. Таму прагляд можа ўтрымліваць толькі палі з індэкса Lucene. Калі вам патрабуецца больш палёў, вы можаце альбо дадаць палі ў індэкс, альбо не выкарыстоўваць выгляд AngularJS. Гэты выгляд вызначаны ў source file web-ui/src/main/resources/catalog/views/default/templates/recordView.html.

Прагляды метададзеных называюцца «фармататарамі». Яны знаходзяцца ва ўбудове схемы, звязанай з метададзенымі, якія вы фарматуеце. Сродкі фарматавання выкарыстоўваюць XSLT або Groovy для пераўтварэння XML у неабходны фармат (html, xml, pdf, json).

Фарматавальнік можна абнавіць з вэб-інтэрфейсу ў кансолі адміністратара, метададзеныя і шаблоны, укладка &#39;фарматары&#39;. На гэтай старонцы вы можаце загружаць, змяняць і праглядаць фарматаўкі.

Пасля стварэння новага фарматэра вам трэба будзе абнавіць код вашага прыкладання, каб можна было візуалізаваць выхад новага фарматэра з вашага прыкладання. Калі мэта фарматавальніка - увесці новы від HTML для метададзеных, вы можаце дадаць спасылку на яго ў source file web-ui/src/main/resources/catalog/views/default/config.js ( searchSettings.formatter.list).
