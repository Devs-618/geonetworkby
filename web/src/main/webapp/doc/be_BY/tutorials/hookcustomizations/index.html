Падключыце свой уласны код¶

У канцы гэтага падручніка вы зможаце наладзіць свой уласны код у GeoNetwork простым і простым для абнаўлення спосабам. Вы павінны спачатку пачаць з падручніка Уводзіны ў GeoNetwork.

  • Аддзяліце свой код
    • Стварыце свой уласны праект
    • Дадайце свой праект
  • Выкарыстанне падзей
    • Уключэнне слухачоў падзей
    • Просты прыклад
    • API GeoNetwork
  • Даданне новага карыстальніцкага інтэрфейсу
    • Стварыце пусты стыль
    • Імпартаваць стыль па змаўчанні
    • Змяніць вынікі пошуку
  • Убудовы схемы
  • Палі пошуку
    • Даданне новага поля
    • Рэзюмэ
    • Павысіць палі

Для наладкі інтэрфейсу глядзіце Наладзьце карыстацкі інтэрфейс GeoNetwork.
