Стварыце новы фарматавальнік¶

Мэты¶

  • назіраць за механізмам фарматавання Groovy

  • разумець канцэпцыі апрацоўшчыкаў, супастаўленняў

  • стварыць невялікі фарматаўшчык

Практыкаванні¶

  1. Дадайце новы фарматавальнік, які адлюстроўвае ўсю тэкставую інфармацыю метададзеных у базавым макеце.

  2. Абнавіце папярэдні фарматавальнік для адлюстравання цэтлікаў тэкставых палёў.

  3. Выкарыстоўвайце сістэму шаблонаў фарматаўца для візуалізацыі выгляду апрацоўшчыка

  4. Дадайце стыль да выгляду фарматаўкі.

  5. Стварыце дрэва для візуалізацыі элементаў

  6. Выкарыстоўвайце пераклад элементаў iso

  7. Выкарыстоўвайце стыль дрэва па змаўчанні

  8. Выкарыстоўваць усе элементы фарматавання прагляду па змаўчанні

  9. Наладзьце выгляд фарматавальніка па змаўчанні

  10. Метады апрацоўшчыкаў перагрузкі

Выпраўленні¶

Перайдзіце на https://github.com/fgravin/core-geonetwork/commits/foss4g

1. Create a new formatter - create a new folder named 'foss4g" in /schemas/iso19139/src/main/plugin/iso19139/formatter - create a new groovy file in this new folder - text information are stored in gco:CharacterString

handlers.add 'gco:CharacterString', {el -> "<div>${el.text()}</div>"}

  1. Дадайце супадзенне і пагуляйце са ўласцівасцямі name і text.

handlers.add select: {el -> !el.'gco:CharacterString'.text().isEmpty()}, {el ->
    "<div><b>${el.name()}</b> - ${el.text()}</div>"
}

  1. Дадайце карыстальніцкую опцыю ў SummaryFactory

  • view.groovy

import iso19139.SummaryFactory

def isoHandlers = new iso19139.Handlers(handlers, f, env)

def factory = new SummaryFactory(isoHandlers, {summary ->
   summary.title = "My Title"
   summary.addCompleteNavItem = false
   summary.addOverviewNavItem = false
   summary.associated.clear()
})


handlers.add name: "Summary Handler",
       select: {it.parent() is it.parent()},
       {factory.create(it).getResult()}
isoHandlers.addDefaultHandlers()

        isoTextEl = { el ->
            "----------- ${oldImpl(el)}"
        }
    }
}
