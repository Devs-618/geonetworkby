Апісанне інфармацыі¶

У гэтым раздзеле апісваецца, як каталагізаваць рэсурсы. Спачатку карыстальнік павінен выбраць стандарт для выкарыстання. У цэлым дзейнічаюць наступныя правілы:

  • `Дублінскае ядро<http://dublincore.org/> `_ звычайна выкарыстоўваецца для партала opendata, спасылкі на публікацыі, справаздачы, ...

  • `ISO 19115/119/139<http://www.iso.org/iso/en/home/store/catalogue_tc/catalogue_detail.htm?csnumber=32557> `_ выкарыстоўваецца для прасторавых рэсурсаў (напрыклад, набораў даных, сэрвісаў, карт)

  • `ISO 19110<http://www.iso.org/iso/en/iso_catalogue/catalogue_tc/catalogue_detail.htm?csnumber=39965> `_ выкарыстоўваецца для табліцы атрыбутаў функцый

Форма рэдактара забяспечвае загрузку дадзеных, графікі, дакументаў, файлаў pdf і любога іншага тыпу кантэнту. Ён падтрымлівае сярод іншага:

  • поўная стандартная падтрымка

  • шматмоўнае рэдагаванне метададзеных,

  • сістэма праверкі,

  • сістэма прапаноў для паляпшэння якасці метададзеных

  • геапублікацыя слаёў для публікацыі слаёў геададзеных у службах OGC (напрыклад, GeoServer)
