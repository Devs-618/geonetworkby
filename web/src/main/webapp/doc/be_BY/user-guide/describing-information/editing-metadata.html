Рэдагаванне метададзеных¶

This topic describes how to use the metadata editor.

Выкарыстанне панэлі інструментаў рэдактара¶

Верхняя панэль інструментаў змяшчае асноўныя дзеянні рэдактара:

Злева направа:

  • Усталюйце катэгорыі метададзеных

  • Усталяваць групу метададзеных

  • Пашыраныя магчымасці (напрыклад, кіраванне версіямі)

  • Validate to run metadata validation

  • Адмяніць, каб адмяніць усе змены, зробленыя з пачатку сеансу рэдагавання

  • Захаваць і зачыніць, каб захаваць і закрыць рэдактар

  • Захаваць, каб захаваць метададзеныя або шаблон. Высадка таксама дазваляе ўсталяваць, ці з&#39;яўляюцца праўкі нязначнымі (г.зн. дата змены метададзеных не будзе абнаўляцца ў выпадку нязначных праўак).

  • Змяніце рэжым прагляду формы рэдактара, уключыце падказкі або пераключыцеся ў пашыраны рэжым рэдактара

Навігацыя па форме¶

To describe the resource, an editor can use the form, which provides different views. By default the editor provides:

  • просты выгляд, які адлюстроўвае ўсе палі з метададзеных

  • поўны прагляд, заснаваны на стандарце метададзеных і забяспечваючы доступ да ўсіх дэскрыптараў са стандартаў

  • an XML view displaying the record as an XML document

Каб пераключыцца з аднаго выгляду на іншы, выкарыстоўвайце наступнае меню:

Once the view is selected, the editor can browse the list of fields to populate. He can also use the scroll-spy on the lower right corner to quickly access a section.

The scroll-spy highlights the visible sections, helps user to return to the top of the form and can be minimized if needed.
