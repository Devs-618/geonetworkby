Імпарт новага запісу¶

Рэдактар можа імпартаваць метададзеныя ў файл каталога ў розных фарматах: XML, MEF або ZIP (гл. Фармат абмену метададзенымі (MEF)).

Before you begin¶

The user should have an editor profile to access metadata.

  1. Go to the contribute page and select Import new records.

Using the import new records page, you can:

  • абярыце Загрузіць файл з вашага кампутара і абярыце адзін файл XML або MEF для загрузкі

  • абярыце Капіяваць/Уставіць і скапіруйце дакумент XML у тэкставае поле

  • абярыце Імпартаваць набор файлаў з папкі на серверы і ўсталюйце шлях да папкі на серверы

Каб імпартаваць некалькі файлаў адначасова, выкарыстоўвайце фармат MEF або параметры імпарту з сервера.

  1. After you have defined the type of import, configure the other import settings:

  • Тып файла: пры загрузцы або загрузцы файла з сервера вызначце тып файла для загрузкі. Гэта можа быць XML для імпарту дакумента XML або MEF (эквівалент ZIP) для імпарту фармату MEF.

  • Тып запісу:

  • Use Metadata when loading a normal metadata record

  • Use Template when the loaded metadata record will be used as a template.

  • Апрацоўка ідэнтыфікатара запісу вызначае, як апрацоўваць патэнцыйныя канфлікты паміж UUID загружанага запісу і UUID запісаў метададзеных, якія ўжо ёсць у каталогу. Даступныя 3 стратэгіі:

  • Няма: UUID загружанага запісу не змяняецца. Калі запіс метададзеных з такім жа UUID ужо прысутнічае ў каталогу, вяртаецца паведамленне пра памылку.

  • Перазапісаць метададзеныя такім жа UUID: любы існуючы запіс метададзеных у каталогу, які мае той самы UUID, што і загружаны запіс, будзе абноўлены.

  • Стварыць UUID для ўстаўленых метададзеных: новы UUID уплывае на загружаны запіс.

  • Ужыць пераўтварэнне XSLT дазваляе трансфармаваць запіс, загружаны з дапамогай табліцы стыляў XSLT. Прадугледжаны спіс наканаваных пераўтварэнняў. Выбранае пераўтварэнне павінна быць сумяшчальным са стандартам загружанага запісу (гл. Даданне пераўтварэння XSLT для імпарту).

  • Validate запускае праверку запісу перад яго загрузкай. У выпадку памылкі запіс адхіляецца і паведамляецца пра памылку.

  • Прызначыць да бягучага каталога прызначыць бягучы каталог у якасці крыніцы для запісу, калі файл MEF паказвае іншую крыніцу.

  • Assign to Group вызначае групу загружанага запісу.

  • Прызначыць катэгорыю вызначае лакальную катэгорыю для прызначэння загружанага запісу.

3. Click import to trigger the import. After processing, a summary is provided with the following details:

  • агульная колькасць імпартаваных метададзеных

  • паведамленні пра памылкі

  • калі імпартуецца толькі адзін запіс, прадастаўляецца спасылка на гэты запіс.
