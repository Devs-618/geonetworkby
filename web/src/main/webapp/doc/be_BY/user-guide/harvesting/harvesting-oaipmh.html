OAIPMH Уборка ўраджаю¶

Гэта пратакол збору, які шырока выкарыстоўваецца ў бібліятэках. GeoNetwork рэалізуе пратакол версіі 2.0.

Даданне камбайна OAI-PMH¶

Сервер OAI-PMH рэалізуе пратакол збору, які GeoNetwork, выступаючы ў якасці кліента, можа выкарыстоўваць для збору метададзеных.

Параметры канфігурацыі:

  • Сайт - Параметры, якія апісваюць аддалены сайт.

    • Імя - гэта кароткае апісанне аддаленага сайта. Ён будзе паказаны на галоўнай старонцы збору ўраджаю як назва гэтага асобніка камбайна OAIPMH.

    • URL - URL-адрас сервера OAI-PMH, з якога будуць збірацца метаданыя.

    • Значок - значок для прызначэння сабраных метададзеных. Значок будзе выкарыстоўвацца пры паказе вынікаў пошуку.

    • Выкарыстоўваць уліковы запіс - уліковыя даныя ўліковага запісу для базавай аўтэнтыфікацыі HTTP на серверы OAIPMH.

  • Крытэрыі пошуку - гэта дазваляе выбіраць запісы метададзеных для ўраджаю на аснове пэўных крытэраў:

    • Ад - тут вы можаце ўказаць дату пачатку. Любыя метаданыя, дата апошняй змены якіх роўная гэтай даце або перавышае яе, будуць сабраны. Каб дадаць або змяніць значэнне для гэтага поля, трэба выкарыстоўваць значок побач з тэкставым полем. Гэта поле неабавязковае, таму, калі вы не ўвялі дату пачатку, абмежаванне скідаецца. Выкарыстоўвайце значок, каб ачысціць поле.

    • Да - функцыянуе гэтак жа, як параметр Ад, але дадае абмежаванне канца для пошуку даты апошняга змянення. Будуць сабраны любыя метаданыя, у якіх даныя апошняга змянення меншыя або роўныя гэтым даным.

    • Набор - сервер OAI-PMH класіфікуе метададзеныя ў наборы (напрыклад, катэгорыі ў GeoNetwork). Вы можаце запытаць усе запісы метададзеных, якія належаць да набору (і любога з яго падмностваў), указаўшы тут імя гэтага набору.

    • Прэфікс - «Прэфікс» азначае фармат метададзеных. Прэфікс oai_dc павінен падтрымлівацца ўсімі OAI-PMH-сумяшчальнымі серверамі.

    • Вы можаце выкарыстоўваць кнопку «Дадаць», каб дадаць больш чым адзін набор крытэрыяў пошуку. Наборы крытэрыяў пошуку можна выдаліць, націснуўшы на маленькі крыжык у левым верхнім куце набору.

Note

выпадальны спіс «Наборы пастаўшчыкоў OAI» побач з тэкставым полем Set і выпадальны спіс «Прэфіксы пастаўшчыкоў OAI» побач з тэкставым полем Prefix першапачаткова пустыя. Пасля ўказання URL злучэння вы можаце націснуць кнопку Атрымаць інфармацыю, якая падключыцца да аддаленага сервера OAI-PMH, атрымае ўсе падтрымоўваныя наборы і прэфіксы і запоўніць выпадальныя спісы гэтымі значэннямі. Выбар значэння з любога з гэтых выпадальных спісаў запоўніць адпаведнае тэкставае поле выбраным значэннем.

Заўвагі¶

  • калі вы запытаеце выхадны фармат oai_dc, GeoNetwork пераўтворыць яго ў фармат Dublin Core.

  • калі вы рэдагуеце раней створаны асобнік камбайна OAIPMH, выпадаючыя спісы set і prefix будуць пустымі. Вам трэба яшчэ раз націснуць кнопку атрымання інфармацыі, каб падключыцца да аддаленага сервера і атрымаць інфармацыю аб наборы і прэфіксе.

  • ідэнтыфікатар аддаленага сервера павінен быць UUID. У адваротным выпадку метаданыя могуць быць сабраны, але падчас іерархічнага распаўсюджвання сутыкненні ідэнтыфікатараў могуць пашкодзіць сабраныя метаданыя.
