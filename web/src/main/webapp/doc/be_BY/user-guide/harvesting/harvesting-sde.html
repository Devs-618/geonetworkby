Збор вузла ARCSDE¶

Гэта пратакол збору метададзеных, якія захоўваюцца ва ўсталёўцы ArcSDE.

Даданне камбайна ArcSDE¶

Камбайн ідэнтыфікуе фармат метададзеных ESRI: ESRI ISO, ESRI FGDC для прымянення неабходных xslts для трансфармацыі метададзеных у ISO19139. Параметры канфігурацыі:

  • Ідэнтыфікацыя

    • Імя - гэта кароткае апісанне вузла. Ён будзе паказаны на галоўнай старонцы збору ўраджаю.

    • Група - адміністратар карыстальніка гэтай групы і адміністратар каталога могуць кіраваць гэтым вузлом.

    • Карыстальнік Harvester - карыстальнік, які валодае сабранымі метададзенымі.

  • Расклад - Канфігурацыя раскладу для выканання камбайна.

  • Канфігурацыя для пратаколу ArcSDE

    • Сервер - IP-адрас або імя сервера ArcSde.

    • Порт - сэрвісны порт ArcSde (звычайна 5151) або порт базы дадзеных ArcSde, у залежнасці ад абранага тыпу злучэння, глядзіце ніжэй раздзел Тып злучэння.

    • Імя базы даных - імя асобніка ArcSDE (звычайна esri_sde).

    • Версія ArcSde - версія ArcSde для збору ўраджаю. Мадэль дадзеных, якая выкарыстоўваецца ArcSde, адрозніваецца ў залежнасці ад версіі ArcSde.

    • Тып злучэння.

      • Сэрвіс ArcSde - выкарыстоўвае сэрвіс ArcSde для атрымання метададзеных.

      Note

      Additional installation steps are required to use the ArcSDE harvester because it needs proprietary ESRI Java api jars to be installed.

      ArcSDE Java API libraries need to be installed by the user in GeoNetwork (folder INSTALL_DIR_GEONETWORK/WEB-INF/lib), as these are proprietary libraries not distributed with GeoNetwork.

      Спатрэбяцца наступныя банкі:

      • jpe_sdk.jar

      • jsde_sdk.jar

      dummy-api-XXX.jar неабходна выдаліць з INSTALL_DIR/web/geonetwork/WEB-INF/lib

      • Прамое злучэнне з базай даных - для атрымання метададзеных выкарыстоўваецца злучэнне з базай даных (JDBC). з

      Note

      Прамое злучэнне з базай даных патрабуе скапіравання драйвераў JDBC у INSTALL_DIR_GEONETWORK/WEB-INF/lib.

      Драйверы Postgres JDBC распаўсюджваюцца разам з GeoNetwork, але не для Oracle або SqlServer.

    • Тып базы даных - Тып базы даных ArcSde: Oracle, Postgres, SqlServer. Даступна, толькі калі для тыпу злучэння настроена Прамое злучэнне з базай даных.

    • Імя карыстальніка - Імя карыстальніка для падлучэння да сервера ArcSDE.

    • Пароль - Пароль карыстальніка ArcSDE.

  • Дадатковыя параметры для пратаколу arcsde

    • Праверка запісаў перад імпартам - вызначае крытэрыі адхілення метададзеных, якія з&#39;яўляюцца несапраўднымі ў адпаведнасці з правіламі XSD і schematron.

      • Прыняць усе метаданыя без праверкі.

      • Прымайце дапушчальныя метаданыя XSD.

      • Прымайце дапушчальныя метададзеныя XSD і schematron.

  • Прывілеі - прызначыць прывілеі для сабраных метададзеных.
