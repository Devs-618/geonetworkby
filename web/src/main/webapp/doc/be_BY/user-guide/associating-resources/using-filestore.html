Загрузка ўкладанняў¶

New in version 3.2.

Калі дакументы недаступныя, рэдактары могуць загрузіць далучэнні да запісу метададзеных. Дакумент дададзены ў сховішча файлаў. Сховішча файлаў можа ўтрымліваць файлы любога тыпу.

Каб загрузіць файл, націсніце кнопку і абярыце файл або перацягніце файл на кнопку. Файлы захоўваюцца ў тэчцы ў каталогу дадзеных (гл. Customizing the data directory). На кожную метададзеныя ёсць адна папка, якая змяшчае:

  • Папка public з файламі, даступнымі ўсім карыстальнікам

  • Папка private з файламі, даступнымі ідэнтыфікаванаму карыстальніку з прывілеямі спампоўвання (гл. Прывілеі кіравання)

З файлавага сховішча:

  • пстрыкніце імя файла, каб усталяваць URL для бягучага дакумента, які трэба далучыць

  • пстрыкніце значок вока, каб праглядзець дакумент

  • націсніце на шафку, каб змяніць бачнасць дакумента

  • націсніце крыжык, каб выдаліць файл.

Файл, загружаны такім чынам, будзе экспартаваны ў файл экспарту метададзеных (MEF). Такім чынам, яго URL не будзе аўтаматычна дададзены ў метададзеныя. URL-адрас дадаецца пры далучэнні дакумента да пэўнага элемента ў метададзеных (напрыклад, агляд, справаздача аб якасці, легенда).

Канфігурацыя Filestore¶

Па змаўчанні максімальны памер файла складае 100 Мб. Гэты ліміт усталяваны ў source file /services/src/main/resources/config-spring-geonetwork.xml з дапамогай параметра maxUploadSize.
