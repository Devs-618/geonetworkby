Аналіз і візуалізацыя дадзеных¶

Даныя даступныя з дапамогай сэрвісаў загрузкі (WFS)¶

Усе наборы даных, апісаныя ў запісах метададзеных і даступныя праз службы загрузкі (напрыклад, WFS), могуць быць прааналізаваны для паляпшэння пошуку і візуалізацыі даных. Каталог змяшчае характарыстыкі WFS і індэксуе іх.

Elasticsearch неабходна ўсталяваць і наладзіць (гл. Installing remote index).

Once collected, filtering is available in the map viewer for this layer. The filter interface provides faceting and export capabilities. The map viewer tries to connect to a WFS service by substituting WMS by WFS in the service URL. If a WFS service is responding with a feature type having the same name as the WMS, the filter panel provides the option to index the features. User needs to be identified to run the indexing task.

Пасля праіндэксацыі ўсе карыстальнікі могуць атрымаць доступ да меню фільтрацыі з:

  • поўны тэкставы пошук па ўсіх палях

  • элемент кіравання для адлюстравання або не адлюстравання цеплавой карты

  • фасеты ва ўсіх палях (вылічваецца аўтаматычна дыяпазон фасетаў для лікаў)

  • дзеянне табліцы адлюстроўваць усе значэнні

  • дзеянне фільтра прымяніць фільтры да слаёў WMS з дапамогай SLD

Націсніце на меню табліцы, каб адлюстраваць атрыбуты. Двойчы пстрыкніце радок у табліцы, каб павялічыць аб&#39;ект.

Таксама могуць прымяняцца прасторавыя фільтры.

Настройка тыпу функцыі¶

Па змаўчанні панэль фільтраў ствараецца з індэкса Elasticsearch. Але вы наладжваеце яго для пэўнага тыпу функцыі WFS.

Настройка выконваецца з дапамогай канфігурацыі JSON, устаўленай у раздзел gmd:applicationProfile інтэрнэт-рэсурсу ў метададзеных.

Вось варыянты:

{
"fields": [{
  "name": "PNT_PROF",
  // non disponible actuellement
  "type": "double",
  "fq" : {
    "facet.interval": "PNT_PROF_d",
    "facet.interval.set": ["[0,10]", "(10,10000]"],
   or
    "facet.range": "PNT_PROF_d",
    "facet.range.start": "0",
    "facet.range.end": "10000",
    "facet.range.gap": "300"
   or
    "facet.range": "PNT_PROF_d",
    "facet.range.classes": "5" // Get min, max and compute gap on 5 classes
  }
}, {
  "name": "GRIDCODE"
}, {
  "name": "LABEL",
  "label" : {"fr": "monLabel", "en": "myLabel"}
}],
"tokenize": {"GRIDCODE": ",", "PARCELLE": "/"},
"heatmap": true
}

Ты можаш

  • абмежаваць палі, якія выкарыстоўваюцца для фільтра

  • задайце ўласныя дыяпазоны лікавых значэнняў

  • усталюйце свой ярлык

  • кіраваць токенізаванымі палямі

  • уключыць цеплавую карту
