Управление категориями¶

GeoNetwork имеет концепцию категорий, которые могут быть присвоены документам метаданных, но они не представлены в метаданных. Поэтому при экспорте метаданных категория будет потеряна. Вы можете использовать эти категории для разделения документов на группы без изменения фактического содержания метаданных. Категории можно использовать для фильтрации результатов поиска или ограничения вывода пользовательской конечной точки csw.

Чтобы назначить категорию документу метаданных. Перейдите к форме изменения метаданных и выберите требуемую категорию из выпадающего меню. Затем сохраните свои метаданные.

Чтобы изменить доступные категории в каталоге, на странице администратора откройте «системы классификации», а затем вкладку «категория».

Note: If you add or modify categories, they may not obtain an appropriate icon. These icon are managed in /catalog/style/gn_icons.less. In this file category-classes are mapped to font-awesome variables that map to a certain font-awesome icon.
