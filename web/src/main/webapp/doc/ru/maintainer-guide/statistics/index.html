Настройка статистики поиска/контента¶

Поскольку статистика поиска и контента GeoNetwork 3.4 хранится в `ElasticSearch<https://www.elastic.co/products/elasticsearch/> `_ используя `Кибана<https://www.elastic.co/products/kibana> `_ инструментальные панели, чтобы визуализировать их в приложении администрирования GeoNetwork.

В этом руководстве описывается конфигурация, необходимая для интеграции ElasticSearch/Kibana в GeoNetwork для хранения и визуализации статистики поиска и контента.

GeoNetwork 3.8.x supports ElasticSearch/Kibana 7.2, other versions may not work properly.

Примечание

В этом руководстве не представлена настройка производственного уровня для ElasticSearch/Kibana. Пожалуйста, обратитесь к документации ElasticSearch/Kibana, чтобы выполнить правильную настройку/настройку для производственной среды.
