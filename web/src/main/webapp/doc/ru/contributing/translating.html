Перевод приложения¶

Чтобы перевести клиентский интерфейс, перейдите в «GeoNetwork на Transifex».<https://www.transifex.com/geonetwork/core-geonetwork/> `_. Только справочный язык (например, en) ДОЛЖЕН быть обновлен на github. Все остальные переводы ДОЛЖНЫ быть на Transifex.

Основными файлами для перевода являются файлы Angular UI:

После перевода новый язык необходимо добавить в приложение.

  • Добавьте файлы в клиентское приложение в source file web-ui/src/main/resources/catalog/locales

  • Зарегистрируйте новый язык в source file web-ui/src/main/resources/catalog/js/CatController.js

// Add the language here
module.constant('gnLangs', {
  langs: {
    'eng': 'en',
    'dut': 'du',
    'fre': 'fr',
    'ger': 'ge',
    'kor': 'ko',
    'spa': 'es',
    'cze': 'cz'
  },


// Lang names to be displayed in language selector
$scope.langLabels = {'eng': 'English', 'dut': 'Nederlands',
  'fre': 'Français', 'ger': 'Deutsch', 'kor': '한국의',
  'spa': 'Español', 'cze': 'Czech'};

To automatically retrieve & update translations files, the script source file web-ui/download-from-transifex.sh can be used. To use the script you have to join the translation team on Transifex.

Перевод стандарта¶

Чтобы перевести стандарт, см. папку схемы (source file schemas/iso19139/src/main/plugin/iso19139/loc). Эти файлы используются в редакторе и средствах форматирования для перевода стандартных элементов, предоставления текста справки, списка рекомендуемых значений, …
