Расширьте свой собственный вид¶

Теперь мы создали нашу структуру с базовым контентом, мы хотим создать настоящую страницу каталога.

Для этого вы можете включить содержимое представления по умолчанию или создать собственное приложение поверх основных компонентов GeoNetwork.

Компоненты GeoNetwork¶

Дизайн пользовательского интерфейса основан на общих виджетах, логике и компонентах. В терминах AngularJS мы говорим о «сервисах» и «директивах».

Все компоненты хранятся в папке `components <https://github.com/geonetwork/core-geonetwork/tree/develop/web-ui/src/main/resources/catalog/components> `_

  • Main page useful components

Основной поисковый ввод

<input type="text"
       class="form-control input-lg"
       id="gn-any-field"
       data-ng-model="searchObj.params.any"
       data-ng-model-options="modelOptionsForm"
       placeholder="{{'anyPlaceHolder' | translate}}"
       data-ng-keyup="$event.keyCode == 13 && triggerSearch()"
       data-typeahead="address for address in getAnySuggestions($viewValue)"
       data-typeahead-loading="anyLoading" class="form-control"
       data-typeahead-min-length="1"
       data-typeahead-focus-first="false">

Панель фасетов

<div data-ng-show="searchResults.records.length > 0"
     data-gn-facet-dimension-list="searchResults.dimension"
     data-params="searchObj.params"
     data-facet-type="facetsSummaryType"
     data-current-facets="currentFacets">
</div>

Диспетчер оповещений

<div data-gn-alert-manager=""></div>

Список результатов

<div data-ng-show="searchResults.records.length > 0"
     data-gn-results-container=""
     data-search-results="searchResults"
     data-template-url="resultTemplate"
     data-map="searchObj.searchMap"></div>

Географический поиск

<div data-gn-map-field="searchObj.searchMap"
     data-gn-map-field-geom="searchObj.params.geometry"
     data-gn-map-field-opt="searchObj.mapfieldOption"
     class="gn-search-map">

  • Основные компоненты карты

    • wmsimport

    • ввод для поиска по карте

    • панель рисования

    • панель измерения

    • панель печати

    • менеджер слоев

Таким образом, вы можете повторно использовать все эти компоненты в своих шаблонах для создания нужного представления.

Стили¶

Важно

Вы должны импортировать стиль по умолчанию и добавлять пользовательские правила CSS вместо того, чтобы создавать свои собственные с нуля.

В gn_search_geoportal.less импортируйте меньше файлов по умолчанию.

  • Импорт общих стилей поиска (если вы начинаете новый вид с нуля)

@import "../../../style/gn_search.less";

  • Импортировать стили поиска представления по умолчанию (если вы основываете свое представление на основе представления по умолчанию)

@import "../../default/less/gn_search_default.less";

Затем в файле less вы можете начать добавлять новые правила.

.gn-resultview .list-group-item {
    background-color: red;
    &:hover {
    background-color: orange;
    }
}
