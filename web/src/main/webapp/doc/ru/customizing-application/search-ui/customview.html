Создайте свой собственный вид¶

Если вы предпочитаете не редактировать общие файлы по умолчанию или вам нужен действительно другой пользовательский интерфейс, лучший способ — создать собственное пользовательское представление.

Добавить новый вид¶

Как было показано в предыдущем разделе, вам потребуется настроить несколько файлов для загрузки нового представления пользовательского интерфейса.

  • создайте новую папку в web-ui/src/main/resources/catalog/views/.

  • добавьте templates/index.html в эту папку.

  • добавьте файл javascript, загружающий ваш основной модуль AngularJS

  • добавить меньше файлов

Настройте свой вид¶

Допустим, мы хотим создать собственное представление с именем «геопортал».

  • Вам нужно будет создать новую папку web-ui/src/main/resources/catalog/views/geoportal.

  • Добавьте новый файл web-ui/src/main/resources/catalog/views/geoportal/templates/index.html, содержащий

<div class="geoportal">Welcome to geoportal</div>

  • Добавьте новый файл web-ui/src/main/resources/catalog/views/geoportal/search.js, содержащий

(function() {

  goog.provide('gn_search_geoportal');
  goog.require('gn_search');

  var module = angular.module('gn_search_geoportal', ['gn_search']);

})();

Примечание

Вы должны включить модуль gn_search.

  • Добавьте новый файл web-ui/src/main/resources/catalog/geoportal/views/geoportal/less/gn_search_geoportal.less, содержащий

.geoportal {
    font-size: 36px;
    font-weight: bold;
}

Запустите свой вид¶

Чтобы увидеть ваше новое представление, вы можете запустить сервис catalog.search с новым представлением параметров: http://localhost:8080/geonetwork/srv/eng/catalog.search?view=geoportal

Or, go into admin settings page and set the system/ui/defaultView (User interface configuration -> Choose the user interface to use) to geoportal.
