Расширенная конфигурация¶

Конфигурация тайм-аута сеанса пользователя¶

Время ожидания сеанса по умолчанию установлено на 35 минут (см. Сеанс пользователя). Этот тайм-аут можно настроить в WEB-INF/web.xml, изменив значение тайм-аута сеанса (время указано в минутах):

<session-config>
    <session-timeout>35</session-timeout>
</session-config>

Для разработчиков тайм-аут определяется как свойство сборки и может быть настроено в фильтрах. См. source file web/src/main/filters/dev.properties#L20
