Версия 3.4.0¶

Новые возможности¶

  • Конфигурация пользовательского интерфейса на странице администрирования и клиентский API JS<https://github.com/geonetwork/core-geonetwork/pull/1848> `_

  • Многоязычная поддержка теперь доступна для администратора тезауруса<https://github.com/geonetwork/core-geonetwork/pull/2094> `_, подшаблоны и URL. Улучшено индексирование ISO19139.

  • `Легко каталогизируйте свои карты<https://github.com/geonetwork/core-geonetwork/pull/2081> `_

  • `Корзина и поддержка выбора, сохраненного пользователем<https://github.com/geonetwork/core-geonetwork/pull/1920> `_

  • Редактор/Поддержка кодирования полигонов

  • Редактор / Улучшенная локальная поддержка XLink

  • Редактор / Добавьте виртуальный тезаурус, чтобы легко подобрать ключевые слова из всего тезауруса.

  • Редактор/Поддержка подшаблона экстента и формата

  • Редактор / Улучшенная конфигурация (например, боковая панель, встроенная директива, …)

  • Подшаблон / `Инициализировать подшаблон экстента из шейп-файлов<https://github.com/geonetwork/core-geonetwork/pull/2096> `_

  • Подшаблон теперь может иметь статус проверки

  • Панели мониторинга и статистика теперь используют Elasticsearch и Kibana

  • `Индексирование функций WFS с использованием Elasticsearch<https://github.com/geonetwork/core-geonetwork/pull/1981> `_

  • Исправить проблему с памятью карты сайта

  • и множество улучшений в пользовательском интерфейсе
