Настройка вида по умолчанию¶

Следуйте :ref:`инструкциям<customview> ` и создайте свое первое пользовательское представление.

Упражнения¶

  1. Создайте новое представление почти без содержимого

  2. Сделайте новый вид похожим на вид по умолчанию

  3. Создайте свой собственный шаблон списка результатов

  4. Сделайте URL основных шаблонов настраиваемыми.

  5. Put the search map upon facets

  6. Давайте быстро создадим новую страницу поиска
