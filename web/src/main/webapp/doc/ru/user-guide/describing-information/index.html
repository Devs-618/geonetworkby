Описание информации¶

В этом разделе описывается, как каталогизировать ресурсы. Во-первых, пользователю необходимо выбрать стандарт для использования. В целом применяются следующие правила:

  • Дублинское ядро<http://dublincore.org/> `_ обычно используется для портала открытых данных, ссылки на публикацию, отчет, …

  • `ИСО 19115/119/139<http://www.iso.org/iso/en/home/store/catalogue_tc/catalogue_detail.htm?csnumber=32557> `_ используется для пространственных ресурсов (например, наборов данных, сервисов, карт)

  • `ИСО 19110<http://www.iso.org/iso/en/iso_catalogue/catalogue_tc/catalogue_detail.htm?csnumber=39965> `_ используется для таблицы атрибутов объекта

Форма редактора обеспечивает загрузку данных, графики, документов, pdf-файлов и любого другого типа контента. Он поддерживает среди прочего:

  • полная стандартная поддержка

  • многоязычное редактирование метаданных,

  • система проверки,

  • система предложений для улучшения качества метаданных

  • геопубликация слоев для публикации слоев геоданных в сервисах OGC (например, GeoServer)
