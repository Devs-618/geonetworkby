Сборщик GeoNetwork 2.0¶

В GeoNetwork 2.1 представлен новый мощный механизм сбора данных, который несовместим с каталогами на основе GeoNetwork версии 2.0. Старые серверы 2.0 по-прежнему могут собирать метаданные с серверов 2.1, но для сбора метаданных с сервера версии 2.0 требуется этот тип сбора. В связи с тем, что GeoNetwork 2.0 был выпущен более 5 лет назад, этот тип сбора данных устарел.
