Сбор урожая OAIPMH¶

Это протокол сбора данных, который широко используется в библиотеках. GeoNetwork реализует версию 2.0 протокола.

Добавление харвестера OAI-PMH¶

Сервер OAI-PMH реализует протокол сбора данных, который GeoNetwork, выступая в роли клиента, может использовать для сбора метаданных.

Варианты конфигурации:

  • Сайт — параметры, описывающие удаленный сайт.

    • Имя — это краткое описание удаленного сайта. Оно будет отображаться на главной странице сбора урожая как имя этого экземпляра харвестера OAIPMH.

    • URL — URL-адрес сервера OAI-PMH, с которого будут собираться метаданные.

    • Значок — значок, назначаемый собранным метаданным. Значок будет использоваться при отображении результатов поиска.

    • Использовать учетную запись — учетные данные для базовой HTTP-аутентификации на сервере OAIPMH.

  • Критерии поиска — позволяет выбирать записи метаданных для сбора урожая на основе определенных критериев:

    • От — здесь можно указать дату начала. Будут собраны любые метаданные, дата последнего изменения которых равна или превышает эту дату. Чтобы добавить или изменить значение для этого поля, вам нужно использовать значок рядом с текстовым полем. Это поле является необязательным, поэтому, если вы не укажете дату начала, ограничение будет снято. Используйте значок, чтобы очистить поле.

    • Until — работает так же, как параметр From, но добавляет конечное ограничение к поиску по дате последнего изменения. Будут собраны любые метаданные, данные последних изменений которых меньше или равны этим данным.

    • Set — сервер OAI-PMH классифицирует метаданные по наборам (как категории в GeoNetwork). Вы можете запросить все записи метаданных, принадлежащие набору (и любому из его подмножеств), указав здесь имя этого набора.

    • Префикс — «Префикс» означает формат метаданных. Префикс oai_dc должен поддерживаться всеми серверами, совместимыми с OAI-PMH.

    • Вы можете использовать кнопку «Добавить», чтобы добавить несколько наборов критериев поиска. Наборы критериев поиска можно удалить, щелкнув крестик в левом верхнем углу набора.

Примечание

раскрывающийся список «Наборы поставщиков OAI» рядом с текстовым полем Набор и раскрывающийся список «Префиксы поставщиков OAI» рядом с текстовым полем Префикс изначально пусты. После указания URL-адреса подключения вы можете нажать кнопку Получить информацию, которая подключится к удаленному серверу OAI-PMH, извлечет все поддерживаемые наборы и префиксы и заполнит раскрывающиеся списки этими значениями. Выбор значения из любого из этих раскрывающихся списков заполнит соответствующее текстовое поле выбранным значением.

Примечания¶

  • если вы запросите выходной формат oai_dc, GeoNetwork преобразует его в формат Dublin Core.

  • когда вы редактируете ранее созданный экземпляр комбайна OAIPMH, выпадающие списки set и prefix будут пустыми. Вы должны снова нажать кнопку получения информации, чтобы подключиться к удаленному серверу и получить информацию о наборе и префиксе.

  • идентификатор удаленного сервера должен быть UUID. В противном случае метаданные могут быть собраны, но во время иерархического распространения конфликты идентификаторов могут повредить собранные метаданные.
