Сбор услуг CSW¶

Этот сборщик подключается к удаленному серверу CSW и извлекает записи метаданных, соответствующие указанным параметрам запроса.

Добавление комбайна CSW¶

На рисунке выше показаны доступные варианты:

  • Сайт — параметры удаленного сайта.

    • Имя — это краткое описание удаленного сайта. Оно будет отображаться на главной странице сбора урожая как имя этого экземпляра харвестера CSW.

    • URL-адрес службы — URL-адрес документа о возможностях сервера CSW, который необходимо собрать. например. http://geonetwork-site.com/srv/eng/csw?service=CSW&amp;request=GetCabilities&amp;version=2.0.2. Этот документ используется для обнаружения расположения служб, которые нужно вызывать для запроса и извлечения метаданных.

    • Значок — значок, назначаемый собранным метаданным. Значок будет использоваться при отображении собранных записей метаданных в результатах поиска.

    • Использовать учетную запись — учетные данные для базовой HTTP-аутентификации на сервере CSW.

  • Критерии поиска. С помощью кнопки «Добавить» вы можете добавить несколько критериев поиска. Вы можете запрашивать только те поля, которые распознаются протоколом CSW.

  • Опции — параметры расписания.

  • Опции — специальные опции сбора урожая для этого комбайна.

    • Проверить — если этот флажок установлен, метаданные будут проверены после извлечения. Если проверка не проходит, метаданные будут пропущены.

  • Права — назначьте права доступа к собранным метаданным.
