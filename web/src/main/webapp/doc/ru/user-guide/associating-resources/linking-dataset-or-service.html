Связывание набора данных с сервисом¶

Метаданные в наборе данных могут быть связаны с метаданными в службах.

План

Добавьте больше деталей о том, как/почему/кодировании

<srv:operatesOn uuidref="b795de68-726c-4bdf-a62a-a42686aa5b6f"
                xlink:href="http://metawal.wallonie.be/geonetwork/srv/fre/csw?service=CSW&amp;request=GetRecordById&amp;version=2.0.2&amp;outputSchema=http://www.isotc211.org/2005/gmd&amp;elementSetName=full&amp;id=b795de68-726c-4bdf-a62a-a42686aa5b6f"/>

В записи метаданных службы нажмите кнопку «Ссылка на набор данных», чтобы открыть селектор записей, в котором перечислены все наборы данных в каталоге. При желании имя слоя можно определить вручную или из списка слоев, полученных из сервиса (с помощью GetCapabilities).

В записи метаданных набора данных нажмите кнопку «Связать с сервисом», чтобы открыть селектор записей, в котором перечислены все сервисы в каталоге.

При подтверждении ассоциации обе записи будут затронуты изменением (в зависимости от привилегий пользователя, т. е. текущему пользователю может быть не разрешено изменять целевую запись).
