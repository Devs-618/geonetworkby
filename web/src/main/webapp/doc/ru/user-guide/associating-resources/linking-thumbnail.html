Иллюстрация с обзором¶

Чтобы помочь пользователю идентифицировать интересующую запись метаданных, вы можете создать графический обзор (или миниатюру) в виде изображения и прикрепить его к записи метаданных. Например, если ваша запись метаданных описывает некоторый набор географических данных, тогда графический обзор может быть изображением карты с легендой, созданной картографической веб-службой OGC.

С записью можно связать один или несколько обзоров.

Миниатюры отображаются в результатах поиска и представлении метаданных:

На панели «Связанные ресурсы» нажмите кнопку «Добавить документ», чтобы открыть мастер «Связать онлайн-ресурс с текущими метаданными». Миниатюры могут быть добавлены с использованием 3 источников:

  • С URL-адреса в Интернете

  • Из файла, прикрепленного к метаданным

  • Из файла, созданного с помощью службы визуализации

Связывание обзора с URL-адресом¶

Когда обзор доступен в виде изображения в Интернете, его можно напрямую связать с записью метаданных. Добавьте необязательное описание для описания изображения:

Если обзор недоступен, прикрепите изображение (png, gif, jpeg) в хранилище файлов (см. Загрузка вложения) и щелкните его имя, чтобы установить URL-адрес:

Вы можете связать столько изображений, сколько хотите. Изображения, связанные в этом режиме, должны быть общедоступны в Интернете, если вы хотите, чтобы обзоры отображались правильно, даже если эта запись метаданных будет собрана другими каталогами.

Создание миниатюры с использованием слоев WMS¶

Если вы зарегистрировали слой WMS в текущей записи метаданных (см. Связывание слоя WMS), его миниатюру можно создать поверх карты базового слоя. Выберите вкладку «Создать миниатюру» и выберите область для печати на миниатюре. Созданный образ добавляется в хранилище файлов.
